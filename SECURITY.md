# Security Policy

## Supported Versions
Only the latest version of Motion-Lyte-JS is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it immediately by sending an email to peterpaultolulope@gmail.com.

Please do not open a public issue for security-related items.

Once reported, we will:
1. Acknowledge the report within 48 hours.
2. Provide an estimated timeframe for a fix.
3. Notify you once the fix is released.
